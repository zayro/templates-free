How to install WordPress

This is a short walkthrough of WordPress installation. This part will be useful if you install WordPress theme for the first time.

Before you start, you must have a server that supports PHP and provide access to a MySQL database. First step is to install WordPress on your server. You must download WordPress ZIP file from http://wordpress.org/download/, extract the files and you’ll get a folder named "wordpress". Now you must upload all files from this folder to your server via FTP client. Content from this folder must be located in the root of the website, so that this location will be the URL of your website.

If you will access your site URL now, you should see only a note: "There doesn’t seem to be a wp-config.php file…".

In next step you must manually create a configuration file. Open the wordpress folder and make a copy of file named wp-config-sample.php. Change name of this copy to wp-config.php and open this file in an editor, it can be a simple editor like Notepad.

Now you must modify this file and add information about your database. You need to have four data pieces to do this:

  • Address of database server — for example: mysql.yourdomain.com
  • Server username — for example: yourusername
  • Server password — for example: 123456789
  • Database name — for example: wpdbname

You must fill these four data pieces into your wp-config.php file. This is how it would look like for our example data:

/** The name of the database for WordPress */
define(‘DB_NAME’, ‘wpdbname’);
/** MySQL database username */
define(‘DB_USER’, ‘username’);
/** MySQL database password */
define(‘DB_PASSWORD’, ‘123456789’);
/** MySQL hostname */
define(‘DB_HOST’, ‘mysql.yourdomain.com’);

Now go back to browser and reload your site that is pointing to your WordPress installation. If all the data from wp-config.php file is correct you will see the installation page:

Just fill in the name of your blog and your email address, and click on Install WordPress button (this data can be changed later). Now you will see the last page of WordPress installation process that shows the default Username, which is always admin, and a randomly generated Password.

You must remember this password or just copy and paste it to a temporary file. After this click on Log In button or enter admin panel by pointing to your browser to http://yourdomain.com/wp-admin. You will be redirected to admin panel login page where you have to enter default Username and randomly generated password. You can change Username and Password when you are logged into admin panel. Just click on Username (admin) in the top right corner of admin panel. And that’s it, now you can install your theme to your WordPress installation. And that’s it, now you can install VELES Theme to your WordPress installation.

You can either choose to upload the theme to your server via FTP (only upload the unzipped folder "VELES" into the folder wp-content/themes/ ) or via wordpress upload function. After copy WP Theme you must login to admin panel (or refresh this panel if you are still logged), go to Themes (under Appearance, on the left WordPress menu) and activate VELES Theme.

Install a WordPress theme

INSTALL

Login to admin panel.

Go to Appearance → Themes and click on the tab Install Themes. Choose upload.

Click on Browse..., select the downloaded theme file and click in Install Now

.

ACTIVATE

After successfull installation click on Activate or go to Appearance → Themes and click on Activate to activate the newly installed theme.

Next you will see a new submenu item Appearance → VELES Options. Use the submenu items VELES Options and menu Portfolio to manage the theme settings.

After activation, you should see the VELES Theme at the top of the admin panel:



Reset your options

Important!
For first time activation, you need to hit Options Reset to store your default options in the database and then hit Save All Changes (Under menu Appearance → VELES Options).

Add a new project to a Portfolio

Log in to your WordPress Administration Panel (Dashboard).

Go to Portfolio section and click Add New.

Fill the following fields:

  • Name the new project
  • Project Description
  • Category of project
  • Set featured image
  • Set image, or up to 3 images fo slider or vido code
  • Set adittional information

Functions are no different from those, that you use when creating a new post or static pages.

If you need a new category, click Add New Category and type the name of the category.

To enable or disable comments on your draft, select or uncheck the option Allow Comments, which is under the description of your new project.

To add an image to your project, click on the link Set featured image and upload a new image or select exists image from Media library.

Once all fields are filled, click on Publish.
