1) Theme Installation - top

  1. Once you download package from themeforest unzip theme package file. The following folders will be in the unzipped file:
    • Versions - theme files that you need upload in your wordpress themes folder.
    • resource\psds - all layered psd sources.
    • resource\fonts - fonts original copy that embedding for cufon use.
    • licensing - license for the theme.
    • documentation - documentation file for theme package.
  2. Before you install the theme, make sure your wordpress version is 3.0 or higher. Then upload the latest version from folder "Versions" find the latest version currently it is 1.6 open the folder 1.6 and copy the versatile folder under that folder and upload it at your WordPress root directory: /wp-content/themes/ through your FTP Client.
    Do not upload the whole folder of the zip package you downloaded from themeforest.
  3. Next Log into your WordPress admin panel and go to Appearance -> Themes subpanel. After that is done, activate the theme as you do activate other themes.

  4. Once activated versatile, you will see the new panels below the menu tabs in Administration Panels.

    • - Does your server have the GD library installed with its version of PHP?

      Versatile uses TimThumb for image resizing. It requires the GD library to be installed. If your thumbnails/images are not working, check with your host to make sure that you have it.

    • - Does your cache folder have the correct permissions?

      You need to give your image cache folder (/themes/versatile/cache/) with 777 permissions. If you don't find the folder then manually create a folder called cache in versatile fodler and give it a permission with 777.

    • - Does the file "timthumb.php" have the correct permissions?

      Make sure the file (/themes/versatile/timthumb.php) has the correct permissions needed for your server. In most cases this is "644" but it may vary depending on where you are hosted. If 644 does not work, try 755 or 777.

    • - Does your server have mod_security setting that are impeding the scripts functionality?

      Some servers may have mod_security settings that will stop the timthumb scripts from working. You should always contact your host if you are having problems to make sure they do not have settings enabled that would stop the script from working.

    • If your server still gets error then remove the following coding in php files find the following coding where it is and then remove the following coding. 

      <?php bloginfo( 'template_directory' ); ?>/timthumb.php?src=

    • NEW - Go to theme options panel and disable timthumb image resizing in bottom of the generals tab.

2) Sliders Setup - top

Versatile contains 2 different sliders Video Slider and Nivo Slider with 2 different stage layouts. Each with the small slider on left and Text on right.

  1. Nivo Slider - Go to slider options tab in versatile theme options panel and select the nivo slider to use with the theme, slide width is 600px width and 300px height.

  2. Video Slider - Go to slider options tab in versatile theme options panel and select the video slider to use with the theme, video width is 550px width and 300px height.

  3. Highlight Text Beside Slider - To use this header highlight option below slider option find the textarea where you can insert the content using the h1 and h4 elements.

  1. Add Slides - Go to slide menu and click on add slide to add posts for the slides.

  2. Featured Image - Locate the Featured Image module on right side and click the Set featured image link. You will see an uploading screen identical to that used when inserting an image into a blog post. Follow the on-screen prompts to choose the image from your computer and upload it or choose it from one of the images already in your Media Library.

  3. Use as featured image - Once the image has successfully been uploaded, look underneath the sizing options, and click on the Use as featured image link. Your Feature image is now set! You can close the Media selector once you’ve chosen the featured image. The image will then appear wherever a featured image is available. Your Feature image is now set! You can close the Media selector once you’ve chosen the featured image. The image will then appear wherever a featured image is available.

  4. Slide Options - After that you have an option that how you want to populate your slide link, link to post, link to category, link to page, or link to manual/external link.

Your Slide options are done now! You can now publish the post and come down to homepage to view the slider, similarly you can add more sliders.

5) Navigation Setup - top

Versatile user wp default navigation, you can use WordPress's Custom Navigation Menu system.

Menu Description/ Subtitle- Toggle the menu and under Title Attribute write the sub title you want to display.

Navigation Sub Menu Dark Background - If you want to use a transparent black background image for the sub menu list then replace the following lines

  1. Find Code - open main.css file and find this code
    .nav ul li a { background:url(../images/opac-80.png) 0 -1px repeat; font-weight:normal; }

    Replace Code - replace that line with this line
    .nav ul li a { background:url(../images/menu_bg_gradient.png) 0 -1px repeat; font-weight:normal;}

  2. Find Code - open colorscheme.css file and find these lines and replace each lines
    .nav ul li a { color:#888; border:1px solid #000; border-bottom:none;} .nav ul li a:hover { color:#999; background: url(../opac-90.png) 0 -1px repeat; } .nav li ul { border-bottom:1px solid #000; }

    Replace Code - replace that lines with these lines
    .nav ul li a { color:#888; border:1px solid #ddd; border-bottom:none;} .nav ul li a:hover { color:#999; background: url(../images/menu_bg_gradient_overlay.png) 0 -1px repeat; } .nav li ul { border-bottom:1px solid #000; }

Go to Appearance -> Menus subpanel and add two menu items as shown in the “Wordpress Official Documentation“.
