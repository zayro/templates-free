What is ASP?
ASP stands for Active Server Pages and runs on Windows NT/2000 servers only. With ASP scripts you can give your website a new look in your content by generating different content for each user and changing the look of your website from user to user. ASP can also output any data from an microsoft access database or any SQL database.
