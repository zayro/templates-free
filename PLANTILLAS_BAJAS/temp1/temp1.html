You can type in text here. Add some text on the button images or on the bar above. This cell is 598 pixels wide by 309 pixels high. If you add too much in here it will break up the graphics on the left. If you want the page to be longer so it will scroll and not break up the graphics then you will have to adjust the last graphic in this table. At the bottom of this html file find the file called spacer.gif. Then you will have to change the height of that graphic as this cell gets longer. You might have to experiment some but you can get the right height so that graphics on the left side will stay together. This was made for 800x600 screen resolution so it will look ok at that size or better at 1024x768. There is a horizontal bar for this template too.
