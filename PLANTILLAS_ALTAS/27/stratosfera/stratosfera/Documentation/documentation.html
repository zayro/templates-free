If you would like to add prettyPhoto, just put data-rel="prettyPhoto" on anchor:

	<a href="#" data-rel="prettyPhoto"><img src="src" alt="" /></a>

If you would like to add magnifying glass icon or video icon to an image, just put class="magnific-glass" for magnifying glass icon or class="magnific-glass video" for video icon on anchor:

	<a href="#" class="magnific-glass"><img src="src" alt="" /></a>

Portfolio Filter: You can add other categories in the filter bar, but pay attention the id you use, you have to put class="id" on portfolio colums. 

<div class="filter">
	<ul>
		<li class="first"></li>
		<li>
			<a href="javascript:void(0);" id="all" class="selected">All Projects</a>   |   
			<a href="javascript:void(0);" id="app">Graphics Projects</a>   |   
			<a href="javascript:void(0);" id="util">Typography Projects</a>
		</li>
		<li class="last"></li>
	</ul>
</div>

	All column with class="util" will appear if you will click on anchor with id="util"
	
<ul id="list">
	<li id="id-1" class="util">Content Here<li>
	<li id="id-3" class="util">Content Here<li>
	<li id="id-5" class="util">Content Here<li>
</ul>

  1. If you would like to edit effect of social image: 
    	/* Socials Big */
	$("ul.social_networks li").hover(function() { 
		var thumbOver = extractUrl($(this).find("span").css("background-image"));
		$(this).find("span").stop().animate({backgroundPosition: '32px 0'}, 150, 'easeOutQuint') ;
	} , function() {

	});
	
	/* Footer social */
	$("ul.social_footer	li a").hover(function() { 
		var thumbOver = extractUrl($(this).find("span").css("background-image"));
		$(this).find("span").stop().animate({backgroundPosition: '16px 0'}, 250, 'easeOutQuint') ;
	} , function() {
		$(this).find("span").stop().animate({backgroundPosition: '0 0'}, 250, 'easeOutQuint') ;
	});
  2. If you would like to change footer flickr border effect go to js/misc.js and modify: 
    //Flickr border
	$(".flickr ul li a").hover(function() { 
		$(this).stop().animate({"border-color": "#d93e39"}, 600, 'swing');
	},
	function() {
		$(this).stop().animate({"border-color": "#202020"}, 600, 'swing');
	});
  3. If you would like to edit effect of featured columns on homepage: 
    //Animate featured background on homepage
	$(".featured_bg").find('img').hover(
		function () {
			$(this).fadeIn(100).animate({top:"-=10px"},100);
		},
		function () {
			$(this).fadeIn(100).animate({top:"+=10px"},100);
		}
	);
  4. If you would like to edit menu fade effect: 
    	$('.main-menu').on('mouseenter', 'li', function() {
			$(this).children('ul').hide().stop(true, true).fadeIn("normal");
		}).on('mouseleave', 'li', function() {
			$(this).children('ul').stop(true, true).fadeOut("normal");
		});
  5. If you want to edit Text Indent of Category on Sidebar : 
    //Categories Text Indent
	
	$(".categories ul li").hover(
		function () {
			if (!($('a', this).hasClass("selected")))
				$('a', this).stop().animate({textIndent: 15}, 800, 'easeOutQuint');
		},
		function () {
			if (!($('a', this).hasClass("selected")))
				$('a', this).stop().animate({textIndent: 0}, 800, 'easeOutQuint');
		}
	);

If you'd like to change the Toogle or Box bar colour, open "07_ShortcodesPage", select from the list of folders, in the layers panel, "Toogle" or "Box" there you’ll find the shape make the necessary adjustments and then save the file e.g. "box_red.png".

If you'd like to change the quote icon, open "07_ShortcodesPage", select from the list of folders, in the layers panel, "Blockquote" there you’ll find the icon make the necessary adjustments and then save the file e.g. "blockquote.png".

If you'd like to change the button colour, open "07_ShortcodesPage", select from the list of folders, in the layers panel, "Buttons" there you’ll find, according to the button size, the Default Button, Hover Button, Pressed Button make the necessary adjustments and then save the file corresponding to your desirable size e.g. "button_big_left_green.png" and "button_big_right_green.png".

If you'd like to change the default avatar picture, open "03_BlogPage_Single_Post_Page.psd" or whatever other .psd wich contains the specific avatar, navigate to the folder in the layer panel make the necessary adjustments and then save the file as e.g. "avatar.png".
