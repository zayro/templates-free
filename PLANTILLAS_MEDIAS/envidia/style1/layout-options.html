Layout Options

Content layout options

This theme allow you to choose the best layout for your content.

Standar layout

The default page is the standard layout used for content pages. It contains a content section and a sub-navigation. This layout can be reversed to display the content and sub-navigation on either side depending on your preference.

To achieve this layout create two div tags and add the class "post" to one and the class "widget" to the other.

Layout with right navigation
<div class="wrapper">
  <div class="sidebar">Sub-navigation here.</div>
  <div class="post">Your content here.</div>
</div>
Layout with left navigation
<div class="wrapper left-sidebar">
  <div class="sidebar">Sub-navigation here.</div>
  <div class="post">Your content here.</div>
</div>

Full width layout

The full page layout uses the full width of the content area. It does not have a sub-navigation or widget making it a good choice for portfolio pages.

This layout only requires a single div tag with the class "full-page".

<div class="full-page">Your content here.</div>

Two Column layout

Requires two div tags with the class "twocolumns".

<div class="twocolumns">First content column.</div>
<div class="twocolumns">Second content column.</div>

<div class="threecolumns">Third content column.</div>
