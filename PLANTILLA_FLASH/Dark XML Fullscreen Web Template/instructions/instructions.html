GETTING STARTED

1. Unzip the xmlFSDarkSite.zip to the directory you want your site to be in.

2. Open the xml folder

SITE VARIABLES

MENU

1. Open the site.xml file. The first variables should look like this:

<menu
myContentWidth="650"
myContentHeight="330"
itemheight="20"
menuwidth="110"
activewidth="120"
yourLogo="images/logo.png"
/>

2. myContentWidth and myContentHeight are the height and width in pixels of the content pages.

3. Item height is the height of each menu button.

4. menuwidth and activewidth are the widths of the menu buttons when it is active and inactive.

5. yourLogo is the path to your logo file. There is a logo.psd file provided if you would like to use it. Save it as a png so it can have a transparent background.

COLORS

1. The next group of vars defines the highlight color (green in this example) looks like this:

<colors>
<highlight hColor="6da61b"/>
</colors>

2. Make sure you use the 6 digit hex color. example 990000. DON'T USE 0x990000 or #990000.

PAGES

1. The next variables are found within the <pages> tags.

2. Each page needs needs to have AT LEAST the following format:

<page pagetype="XXXX" buttontitle="XXXX" pagetitle="XXXX"> </page>

3. The pagetype is what kind of page you want to use. Here are the different types:

textOnly = text only
textImageLeft = text on the right, image on the left
textImageRight = text on the left, image on the right
externalSwf = this is an externally loaded swf or jpg
contactForm = this is the contact page

4. USE MY PAGES AS TEMPLATES FOR YOUR PAGES. Each page type has different variables so just use the ones I already have. You can have as many of each as you want but just copy and paste from mine.

5. buttontitle is the text that shows up on the buttons.

6. pagetitle is the title that shows up at the top of your content pages

7. If your page uses an image you need to define imgSrc (sourse of your image), imageWidth, imageHeight, and borderWidth.

8. If your page has text on it put it in like this:

<page pagetype="XXXX" buttontitle="XXXX" pagetitle="XXXX"><![CDATA[YOUR TEXT GOES HERE]]> </page>

9. Don't forget the <![CDATA[ ]]> tags

10. The contactForm page type works like a page with an image.

MUSIC

1. Should look like this:

<music
useMusic="true"
autoPlayYN="true"
musicPath="music/adg3.com_cloudlessDays.mp3"
/>

2. useMusic needs to be true or false. It detemines if you want to use music or not. If false don't worry about the other variables.

3. autoPlayYN is true or false and determines if you want it to autoplay or wait for the user to push play.

4. musicPath is the path to the mp3 file.

GALLERY

1.Open gallery.xml in the xml folder

THUMBS

1. Should look like this:

<menu
thumbWidth="64"
thumbHeight="48"
/>

2. These two variables are the width and height of the thumbnails. You can make the thumbs whatever size you want but THEY ALL NEED TO BE THE SAME SIZE.

IMAGES

1. Each image should look like this:

<image src="XXX.jpg"><![CDATA[ ]]></image>

2. Each image needs a src defines. example yourimage01.jpg. The images go in your images folder.

3. The thumb should have the same name as the origonal image and be stored in the thumbs folder.

4. make sure the thumb is resized to the size you determined in the thumb variables earlier.

5. If you want a caption put it here:

<![CDATA[YOUR TEXT HERE]]>

6. If you don't want a caption leave it like this:

<![CDATA[]]>

CONTACT

1.Open contact.xml in the xml folder

2. email is the email you want to receive the emails

3. nameTxt, emailTxt, messageTxt, sendTxt and sendingTxt are all just the text that the user sees.

4. The text on the page looks like this:

<![CDATA[YOUR TEXT HERE]]>
