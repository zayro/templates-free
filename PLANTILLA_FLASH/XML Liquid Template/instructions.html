Change Music

1. Save your mp3 music file in the folder labeled "tracks".
2. Then open the file called tracks.xml in notepad or Dreamweaver.
3. Change the paths to reference your new mp3's.
4. The list plays in descending order so list the tracks in the order you want them to play.
5. Also name the artist and track title in the xml.
Example: <song url= "tracks/adg3.com_crypticPsyche.mp3" artist="Flash Den Archive" track="Cryptic Psyche"/>

Email php form

How to Customize the php file

1. Open the file called "sendmail.php" in Dreamweaver and update your email address. You can have more than one email, just seperate with commas.
2. When you upload your website make sure the php file is in your root directory, your webserver must
support php so check that they do.
3. You must also update your domain address path to the php file inside fullbrowser.fla. To do this click on the send button movie clip. Line 15 in the actionscript you'll see :
loadVariables("http://www.yourdomain.com/sendmail.php", this, "POST");
Change the code highlighted in blue to your domain.

To make changes on the look of the contact form such as highlight color etc do the following:

4. Inside "fullbrowser.fla" go to the "contact" frame label, then the Content layer. Click until your inside the contact form movie clip.
5. Inside here you will find an actions layer called "Text Styles" change the settings to skin the form.
6. To change font formatting just click on the text fields and edit there properties.

Example of settings:
- normal_border = 0xFFFFFF;
- select_border = 0xFFFFFF;
- normal_background = 0xFFFFFF;
- select_background = 0x666666;
- select_color = 0xFFFFFF;
- normal_color = 0x666666;

7. Now just publish your flash file and upload everything to your webserver. You can only test the form from a webserver, it won't work on your local computer.

 

Image Gallery

1. The photo slide show currently has 8 galleries. You can add as many galleries as you want, or delete galleries you don't need.
2. The images for each gallery are in the folders labeled galleries/images1-10.
3. Just overwrite the files in each of these folders with yours to update.
4. To add more images first save them in the correct folder for your gallery.
5. Open the corresponding gallery.fla file inside _source folder.
6. Click inside the "Gallery" movie clip
7. Click on the actions layer and open your actions panel.
8. You will see the 2 arrays clearly commented. One for the images and one for the image titles.
9. Just add the paths to your new images, and add the titles for each image in these arrays.

Create an additional Gallery

10. To create another gallery open gallery8.fla
11. Save as gallery9.fla
12. Go to file/Publish settings and change the flash swf path to "../gallery9.swf"
13. Follow the steps described above to reference your new image files and name the titles.
14. Then compile (ctrl-enter)
15. Do the same for each additional gallery.

Create more buttons to load new galleries

16. After you create new galleries you will also need to create additional buttons to load them.
17. Inside fullbrowser.fla click inside the "slideshow" movieclip
18. You will see a series of buttons.
19. Duplicate one of these buttons, position it where you want and give it a unique instance name.
20. Then go to the actions layer, you will see the following code:

nav1.onRollOver = incoming;
nav1.onRollOut = begining;
nav1.labelMC.labelTxt.text = "GALLERY 1";
nav1.onPress = function(){
loadGallery.loadMovie("gallery1.swf")
};

Duplicate this code and change nav1 to your new buttons instance name. Also change gallery1.swf to the name of your new swf file. Lastely change text="GALLERY 1" to the name of your new gallery.

 
 

Video Gallery

1. First save your flv movies inside the "flv" folder.
2. Inside fullbrowser.fla click inside the videoGallery movieclip.
3. You will see frame labels labeled vid1-vid15.
4. Beneath each label you will see an flv movieclip on the layer labeled "FLV Players".
5. Inside each player clip you will see the code commented where you change the path to your movie.
6. Inside each player there is a layer called still, click inside the moviclip on this layer and you will be able to update the still image to your own.

Video Thumbnails

7. Inside the videoGallery moviclip you will see a layer called FLV Thumbnails, click inside each movieclip and update with your image thumbnail.
