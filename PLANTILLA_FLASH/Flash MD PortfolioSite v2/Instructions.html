How to deploy the site:

I have tried my best to make the site as easily customizable as possible without adding too much complexity. Here's step-by-step instructions on how to get the site up and running!

  1. In a text editor of your choice, open mail_script.php. You can find this file at: deploy/php and change the email address to your own.
  2. Next, edit the XML files inside the XML folder (i.e. port1.xml). You can find these files at: deploy/xml - change all portfolio information, including file names, titles, urls, and descriptions.
    Note: I have changed the way I'm handling these XML files, so just add UP TO 20 items per file and the template will take care of the rest.
  3. Open main.as. You can find this file at: fla/as. Change highlight color, border size, sub nav options, and copyright notice to something of your choice.
    Further, if you're ready to deploy the site, you MUST set testMode to false!
  4. If you wish to add buttons to the site, open site_main.as, which can be found at fla/as, and follow the example on lines 159-165 - starting with // ABOUT BUTTON.
  5. (optional) - If you need to change the path to the class files (mosessupposes), open Publish settings, and click on Actionscript 2.0 to change your class path.
  6. Save and publish your movie.
  7. Upload ALL FILES INSIDE deploy FOLDER to your site and test.
