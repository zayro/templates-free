How do I edit the file?

Go inside the "deploy" folder and open up "settings.xml" in your favorite text editor. Simply fill in the values with your new text and colors. Afterwards, refresh the portfolio page and the site will be updated.

How do I edit the portfolio items?

To edit the portfolio items you will need to open up "selected_work.xml" in the "deploy" folder and update the text. To add your own images you will need to save them to the folders "thumbs" and "images" inside the "selected_work" folder.

How many portfolio items can I have?

You can have a maximum of 18 portfolio items with up to 5 images for each item.

What are the correct JPG dimensions to use for portfolio items?

Thumbnail images should be sized 144x60 pixels and portfolio piece images should be sized 475x260 pixels.

How do I add the "Launch Project" links?

Edit the "selected_work.xml" file in the "deploy" folder and update the "link" property to the URL of your project. To link to a live URL outside your domain follow the example in selected_work > projects > sample > index.html.

What fonts are used?

The fonts used are VAG Rounded and Georgia Italic.

How do I change the background color of the site?

Edit the "background_color" property in "settings.xml" and also be sure to change the color in the "index.html" file -- there should be two places to change it -- one in the CSS at the top and the other in the SWF insertion code towards the bottom.

How do I use my own logo?

To use your own logo open up "settings.xml" and set "use_own_logo" to true. Then set "your_logo_file" to the location of your logo file.

How can I customize additional things that aren't in the XML file?

Open up the various .as files in the "src" folder -- usually at the top of the file there will be a list of variables defined that you can customize. Additionally, you can directly edit the portfolio.fla and contact.fla files.
