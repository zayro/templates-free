Adding portfolio works

In this one, I'm going to show you how to add new portfolio works. At the moment there are 5, but I'm sure you'll need more. I'm sure you'll have more clients, so we gotta know how to add new portfolios. Let's get started!

Introduction

Let's open the swf directory and let's have a look inside. There are all sorts of swf,fla,xml files and directories. What's the deal with them? I'm going to explain now. Everytime we press a menu button, it loads an external swf file from this swf directory. If we talk about the portfolio section, that means that we will have a lot of swf files (in case we have a big portfolio). Right now, there are only 5 portfolio swfs. If you had more than 5 clients (and im sure you did) we will have to duplicate one of this 5 swf files and make the 6th one, the 7th and sooo on. It's simple! Don't worry! ;)

Step 1

Every project from within the portfolio section is related to 4 files from the swf directory. For example, Website 1 from the menu, is related to project1.fla,project1.swf,project1.xml and project1.thumbs directory from the swf directory. We gotta select this 4 files, copy them using Ctrl+C and paste them somewhere else Ctrl+V until we edit them (anywhere)

Step 2

We want to make a new portfolio work (the 6th one) and our files are named project1 bla bla. Let's rename all our files to project6.fla,project6.swf,project6.xml and project6.thumbs

Step 3

Now let's open the project6.xml file and edit it a little in order to work correctly. Change all the paths from

path = "swf/project1.thumbs/"

to

path = "swf/project6.thumbs/"

Now flash will know to search for the pictures inside the new thumbs directory that we created called project6.thumbs.

Step 4

Now open the project6.fla file and let's tell flash that we changed the name of the xml file to project6.xml. Open the ActionScript pan by pressing F9, go to line 30 and let's edit that as shown in the picture:

Save changes and export the movie by pressing Ctrl+Enter Don't worry about errors or that nothing can be seen. It's ok! I'll explain that later ;)

Step 5

Now crop you new pictures to 600x400 and replace existing ones from the project6.thumbs directory.

Step 6

Now cut all these new 4 files and paste them inside the swf directory among with the other files.

Step 7

And now that everything is ready, the last thing that we need to do is tell the menu to open our new portfolio work. Alright! Open the items.xml file from the root directory. Let's tell the menu that if we press Website6 inside the Projects > Websites section, he should open our project6.swf file (shown in the picture). Then save your changes inside the items.xml and let's test our site! It's working!

Conclusion

The whole procedure may look "hard to do" at the beginning, but once you get the steps(and it's logic) I'm sure you'll say "adding new portfolios is a piece of cake!". :)
