Setting the contact form

In this one, I'm going to show you how to set the contact form to send all the data to your email address.

Step 1

Alright! Go to the swf directory and using a text editor, open the mail.php file.

Step 2

Now go to the mail setup section and change my email address with yours. Save and exit! That's all!

Further info

The contact information can be set from within the fla file.
