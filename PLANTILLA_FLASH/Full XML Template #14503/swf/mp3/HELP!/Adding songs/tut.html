Adding songs/mp3's

In this one, I'm going to show you how to add new mp3's inside our player.

Step 1

Alright guys, let's open our root directory and let's see what's all about! In order to add new mp3's, we will concentrate on two things: one is the mp3s directory and the other is the data.xml file. In the mp3s directory we will place all our mp3 files and by editing the data.xml file we will tell flash what to open, from where and what it's title,artist,album name are. Step 2

Now that we know where and what, let's place our new mp3 file inside the mp3s directory. I named it 6.mp3 but you can name it however you want as long as you make the good connection with the XML.

Step 3

We placed our new mp3 where it belongs, now let's open the data.xml file and let's call that new song! Copy and paste the last node and change it's path as shown in the picture. Be sure you also modify it's title, artist, album etc.
